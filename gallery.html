Funny Bunny's Small Race was made in 2 weeks for a final project for a class I was taking at Marshall University. I made the game to start working out a game for an IP I've been thinking about making into a game since I was a kid. It was originally made in Unity but I have since remade and started updating the project in Unreal Engine. I made the rabbit in Maya and animated it in Unity. It was the first time I'd animated as well so, there's that. The game is hard so if you can't make it past the sunset level I understand.

Unity
Maya


Cops N Robbers was a game I made in a about an hour for a game jam my freshman year. The theme of the jam was to only use squares "pixels" and that each square had to be a solid color, but they didn't say I couldn't change it. The hardest part about this one was how limited you were but it was a really good exercise.
